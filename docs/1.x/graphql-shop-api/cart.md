# Cart

In this section, we will see all the APIs which are related to Customer Cart.

## Get Cart

You can get all the  Cart from the Krayin store. You can achieve this job by using the `Cart` API call.

- Headers

  | Key           | Value            |
  | ------------- | ---------------- |
  | Accept        | application/json |

- Single Request Endpoint

  `POST http(s)://example.com/graphql`

::: tip

  If you are using pagination and want to know more info about responses then check the [explanation](./explanation) portion.

:::
