# Orders

In this section, we will see all the APIs which are related to Customer Orders.

## Get Customer Orders

- Headers

  | Key           | Value            |
  | ------------- | ---------------- |
  | Accept        | application/json |

- Single Request Endpoint

  `POST http(s)://example.com/graphql`
