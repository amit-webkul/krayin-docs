## Catalog Rules

In this section, we will see all the APIs which are related to Catalog Rules.

### 1. Get All Catalog Rules

You can get all the Catalog Rules from the Krayin store. You can achieve this job by using the `Catalog Rules` API call.

- Headers

  | Key           | Value            |
  | ------------- | ---------------- |
  | Accept        | application/json |

- Single Request Endpoint

  `POST http(s)://example.com/graphql`

::: tip

  If you are using pagination and want to know more info about responses then check the [explanation](./explanation) portion.

:::

**Note**: You can pass all the related fileds that you want in response.

- Query
  ~~~query
    query catalogRules {
      catalogRules {
        paginatorInfo {
          count
          currentPage
          lastPage
          total

          id
  ~~~
